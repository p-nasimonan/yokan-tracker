# yokan-tracker

VRトラッカー用のKiCadプロジェクトです。

構成の出発点は以下です。

- MCU: XIAO ESP32-C6
- IMU: LSM6DSVモジュール
- 電源: LiPoバッテリー

## Gitでの管理方針

- 回路図、基板、プロジェクト設定はGitで追跡する
- `fp-info-cache` や `*.kicad_prl` のようなローカル状態は追跡しない
- カスタムフットプリントはこのリポジトリで管理する
